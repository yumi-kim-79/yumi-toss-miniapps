## 1. 내 인생 시급 (life-wage)

| 항목 | 값 |
|---|---|
| 한국어 앱 이름 | 내 인생 시급 |
| 영문명 | LifeWage |
| 부제 | 이 커피, 내 인생 30분이었네 |

### 상세 설명

점심 1만원, 커피 5천원, 옷 10만원.
숫자만 보면 그냥 가격이지만, 시급으로 환산하면 이야기가 달라져요.

월 실수령액과 한 달 근무시간만 입력하면 시급이 자동 계산되고,
지출 금액을 입력할 때마다 "내 인생 몇 시간"으로 환산해서 보여줘요.

— 점심 1만원? 시급 2만원이면 30분 일한 값.
— 신발 15만원? 7시간 30분 일한 값.

월별 누적 환산 시간으로 "이번 달 사라진 인생"도 한눈에 볼 수 있어요.
충동 결제 직전, 시간으로 한 번 더 생각해 보세요.

본인이 직접 입력한 데이터만 사용하고, 익명 식별자로만 저장돼요.

### 검색 키워드

`시급`, `월급`, `시간`, `가계부`, `충동소비`, `지출관리`, `노동시간`, `시간가치`, `인생시급`, `점심값`

## 2. 내 카드 장례식 (card-funeral)

| 항목 | 값 |
|---|---|
| 한국어 앱 이름 | 내 카드 장례식 |
| 영문명 | CardFuneral |
| 부제 | 안 쓰는 카드, 이제 보내줄 시간 |

### 상세 설명

지갑 속 카드, 마지막으로 언제 썼는지 기억나세요?

각 카드의 별명·카드사·연회비·마지막 사용일을 등록하면,
1·3·6개월 컷오프로 카드의 상태를 표시해 드려요.

— 활성: 한 달 안에 쓴 카드
— 휴면: 한 달 ~ 3개월
— 위중: 3개월 ~ 6개월
— 사망: 6개월 이상

8개 주요 카드사의 해지 가이드(전화번호·앱 경로)도 함께 안내하고,
이미 떠나보낸 카드와 그동안 새어 나간 연회비를 추모관에서 정리해 줘요.

쓰지도 않는 카드 때문에 매년 빠져나가는 연회비,
조용한 작별을 도와드려요.

### 검색 키워드

`카드`, `휴면카드`, `카드해지`, `연회비`, `지갑정리`, `카드관리`, `카드사`, `해지방법`, `신용카드정리`, `카드추모`

## 3. 경조사 가계부 (gyeongjo-book)

| 항목 | 값 |
|---|---|
| 한국어 앱 이름 | 경조사 가계부 |
| 영문명 | GiftLedger |
| 부제 | 주고받은 마음을 잊지 않게 |

### 상세 설명

결혼식 축의금, 부의금, 돌잔치 봉투.
받았던 사람에게 답례할 때, "얼마였더라" 다시 떠올려야 했던 적 있죠.

경조사 가계부는 주고받은 모든 경조사비를 사람 단위로 정리해 줘요.

— 누구에게, 어떤 일로, 얼마를, 언제 줬는지(받았는지)
— 사람별 자동 합산으로 잔고 표시 (내가 더 받았는지·줬는지)
— 이름 자동완성으로 빠른 입력
— 결혼식·장례식·돌잔치·기타 4종 분류

10년이 지나도 그 사람의 기록은 사라지지 않아요.
주고받은 마음을 잊지 않게, 답례할 때 다시 꺼내 보세요.

본인이 직접 입력한 데이터만 익명으로 저장돼요.

### 검색 키워드

`경조사`, `가계부`, `축의금`, `부의금`, `돌잔치`, `답례`, `결혼식`, `장례식`, `경조사비`, `인간관계`

## 4. 오늘의 환승 동전 (coin-saver)

| 항목 | 값 |
|---|---|
| 한국어 앱 이름 | 오늘의 환승 동전 |
| 영문명 | CoinSaver |
| 부제 | 결제할 때마다 동전이 떨어져요 |

### 상세 설명

6,300원 결제? 1,000원 단위로 올림하면 7,000원.
그 차액 700원이 가상 저금통에 떨어져요.

큰돈 모으기는 부담스럽지만, 100원 200원이라면 가벼워요.
결제 금액만 입력하면 자동으로 차액이 계산되어 누적되고,
🪙 1,000원, 1만원, 5만원, 10만원 마일스톤마다
저금통이 격려 멘트로 응원해 줘요.

— 6,300원 → 700원 동전
— 10,000원 → 딱 떨어져 0원
— 1,001원 → 999원 동전 (오늘의 행운)

가상 동전이라 실제 출금은 안 되지만,
"이만큼 모았네" 보는 재미와
1,000원 단위 결제 습관이 자연스럽게 생겨요.

가벼운 시작이 가장 오래 갑니다.

### 검색 키워드

`적금`, `저금`, `동전`, `잔돈`, `푼돈저금`, `저축습관`, `1000원적금`, `시뮬레이션`, `챌린지`, `가계부`

## 메모

- **V1 마케팅 자산**. 출시 후 사용자 반응 + 토스 콘솔 자동 노출 데이터 보고 V2에서 다듬을 예정.
- 토스 콘솔 등록 시 부제·상세 설명·키워드 글자수/개수 제한이 다를 수 있어 콘솔 가이드 재확인 필요.
- 영문명은 모두 15자 이내, 명사형 (LifeWage 8 / CardFuneral 11 / GiftLedger 10 / CoinSaver 9).
- 4개 앱 모두 토스 정책에 따라 만 19세 이상 대상이며, 비속어·종교·정치적 표현은 사용하지 않음.
